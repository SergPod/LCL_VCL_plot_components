Introduction

This is a set of Delphi/Lazarus components and classes for 2D plot (sgr_vcl.dpk/sgr_lcl.lpk). It is relatively simple. It more is of a gauge than a presentation element. The main features are:

  • Two horizontal and two vertical axes.
  • An auto minimum and maximum for the axes.
  • A swap of the Min and Max axis positions.
  • MoveMinMax method to provide smooth scrolling of axis ticks.
  • Panning and Zoom by mouse.
  • Buffered screen paint, can be off.
  • Copy to clipboard as DIB or as Metafile.
  • Custom Data Series.
  • DrawPlot method to replicate plot image
  • OnDrawField and OnDrawEnd events

The main component is the Tsp_XYPlot. It holds four Tsp_Axis and controls the Data Series. The Tsp_XYPlot and Tsp_Axis provide base plot functionality like auto scale, zoom, and pan, and so on. The Data Series must be derived from Tsp_PlotSeries and implement data storage and how data are painted on the plot canvas. Examples of these components are Tsp_XYLine and Tsp_SpectrLines. A Plot also can have and control components inherited from Tsp_PlotMarker. A Plot marker draws something on the plot using the axis coordinate system. Examples of these components are Tsp_LineMarker and Tsp_ImageMarker.

How to start

Place the plot component and the data series component on the form, set the Plot property of the data series to link it with the plot. Play with the visual properties of the plot and series. There are also some notes of using SGraph. The subdirectory test contains files of simple test programs (Lazarus and Delphi 12), which can be compiled without installing components to the IDE
